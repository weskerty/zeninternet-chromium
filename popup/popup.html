OS Compatibility: Only some operating systems support transparency:

  • macOS (native support)
  • Windows 11 (native mica and acrylic)
  • Windows 11 with mods like Mica for Everyone or DWM blur glass
  • Windows 10 with DWM blur glass (Questionable)
  • Wayland on KDE with Kvin blur
  • GTK Gnome DE (experimental with blur my shell)

Common fixes:

  • Check if transparency enabled in the OS
  • Turn off power saving
  • Turn off "Show accent color on window borders" in Windows
  • Disable conflicting mods like better letterboxing or custom themes
  • Adjust Zen theme contrast slider for more transparency
  • Try re-opening the browser

Each website can have customized features like transparency, hover effects, and layout adjustments.

  • Browse to any website and check the addon popup
  • You'll see available features for that site
  • Toggle individual features on/off as needed
  • Disabling features may require a page reload
  • Use the "Global Feature Controls" in Advanced Settings for global overrides

This can happen if the website's text color is too light or dark against a transparent background.

If you are on a website without a dedicated dark theme, you can try using darkreader addon for better contrast.

  • Toggle website's dark/ light theme manually
  • Try adjusting the Zen theme contrast slider
  • Disable transparency for that site if needed
  • Report the issue to request a specific theme

These modes control how built-in and forced themes and features are applied:

  • Blacklist Mode: Apply styles to all sites except those you manually skip
  • Whitelist Mode: Only apply styles to sites you specifically select

    • Whitelist mode is generally recommended for better performance and control but may require more manual setup for each site.

Force theming applies basic transparency to websites without specific themes:

  • Can be used with the blacklist or whitelist mode
  • Includes dark theme support with darkreader addon for better transparency
  • May cause some visual issues on complex sites

Fallback Background: Use this option for sites with no default background color when transparency doesn't take effect. Enable if skipping the website still leaves it transparent.

You can request themes and report bugs directly from the addon:

  • Use the built-in "Request Theme" button for the active website
  • Click "Bug?" in the footer to report issues of the addon
  • Theme contributions are welcome on GitHub

Unsupported sites: Mozilla addons store, Settings pages, New tab page, PDF viewer, and other restricted pages cannot be themed.
